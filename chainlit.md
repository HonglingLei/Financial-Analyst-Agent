# Financial Analysis Agent

An AI-powered financial research assistant that helps you analyze stocks, understand market trends, and make informed investment decisions.

## About This Project

This agent uses LangChain's framework with OpenAI's GPT-4o-mini to autonomously select and execute financial analysis tools based on your natural language queries. It fetches real-time market data from Yahoo Finance and generates interactive visualizations.

## Features

**Data Analysis:**
- Real-time stock prices and market metrics
- Fundamental analysis (P/E ratios, profit margins, growth rates)
- Company information (sector, industry, business overview)
- Latest news articles about companies
- Multi-stock comparisons

**Visualizations:**
- Interactive candlestick price charts
- Performance comparison charts
- Trading volume analysis
- Customizable time periods (1d, 5d, 1mo, 3mo, 6mo, 1y, 2y, 5y, 10y, ytd, max)

## How to Use

**Example Queries:**
- "What is Apple's current stock price?"
- "Analyze Tesla's fundamentals"
- "Compare NVDA vs AMD performance over 6 months"
- "Show me Microsoft's price chart for the past year"
- "What's the latest news about Google?"
- "Plot trading volume for Amazon over 3 months"

**Tips:**
- Use stock ticker symbols (AAPL, TSLA, MSFT) for best results
- The agent will automatically select appropriate tools for your query
- Expand "Agent Execution" to see which tools were used
- Charts are interactive - hover, zoom, and pan to explore

## Disclaimer

This tool is for **educational purposes only** and should not be considered financial advice. Always conduct thorough research and consult with financial professionals before making investment decisions.
